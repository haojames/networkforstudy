🐧 Ubuntu Server Internet Configuration

Step-by-step guide to configure Ubuntu networking for Internet access

🎯 Objective

Configure Ubuntu Server networking to ping the Internet and install required packages (DHCP Server).

🔍 Check Current Status

ip a

The network interface (usually ens3 or eth0) is UP but has no IP address assigned.

🔧 Static IP Configuration with Netplan

Step 1: Identify Netplan config file

ls /etc/netplan/

Step 2: Edit configuration file

sudo nano /etc/netplan/01-netcfg.yaml

Step 3: Add configuration

network:
  version: 2
  renderer: networkd
  ethernets:
    ens3:                      # Replace with your interface name
      dhcp4: no
      addresses:
        - 172.16.10.10/24
      gateway4: 172.16.10.2
      nameservers:
        addresses:
          - 8.8.8.8
          - 8.8.4.4

Step 4: Apply configuration

sudo netplan apply

Step 5: Verify IP assignment

ip a show ens3
✅ Success! Ubuntu now has IP 172.16.10.10/24

🌐 FortiGate Firewall Policy

To allow Ubuntu to access the Internet, add a NAT policy on FortiGate:

config firewall policy
    edit 1
        set name "Ubuntu-to-Internet"
        set srcintf "port3"
        set dstintf "wan1"
        set srcaddr "all"
        set dstaddr "all"
        set action accept
        set schedule "always"
        set service "ALL"
        set nat enable
    next
end
📌 Explanation: set nat enable allows Ubuntu to use FortiGate's WAN IP address to access the Internet.

✅ Connectivity Tests

1. Ping to Gateway (FortiGate)

ping 172.16.10.2

✅ Expected: 64 bytes from 172.16.10.2: icmp_seq=1 ttl=64 time=0.8ms

2. Ping to Internet

ping 8.8.8.8

✅ Expected: 64 bytes from 8.8.8.8: icmp_seq=1 ttl=55 time=15.2ms

3. Ping with Domain Name

ping google.com

✅ Expected: Domain resolution works, ping successful

4. Update Package Lists

sudo apt update

✅ Expected: Repository lists update successfully without errors

🎉 Success! Ubuntu now has full Internet access to install packages.

📦 Install DHCP Server

sudo apt install isc-dhcp-server -y

🐞 Troubleshooting

ErrorSolution
